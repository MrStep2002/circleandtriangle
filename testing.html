1. Headings & Paragraphs

<h1> - <h6> Headings come in six sizes, from biggest to smallest. The <p> tag wraps a paragraph of text — like this one you're reading right now.

This is an <h3> heading

And this is a paragraph below it. You can make text bold with <strong>, or emphasized with <em>. You can even highlight it with <mark>.

2. Links & Images

<a> The anchor tag creates a hyperlink. For example, here's a link to MDN's HTML docs — a great place to learn more.

<img> The image tag displays a picture. Notice it's "self-closing" — it doesn't need a separate closing tag.

3. Lists

<ul> An unordered list uses bullet points:

  • Apples
  • Oranges
  • Bananas

<ol> An ordered list uses numbers:

  1. Wake up
  2. Drink coffee
  3. Write some HTML

4. Quotes

<blockquote> Use a blockquote when you're quoting someone:

"The best way to predict the future is to invent it." — Alan Kay

5. Tables

<table> Tables organize data into rows and columns.

TagWhat it does
<tr>Defines a table row
<th>A header cell
<td>A regular data cell

6. Forms & Buttons

<input> <button> Forms let users type things and click things.





7. Hidden Surprises

<details> Click the arrow to reveal more. This is a great way to hide extra info.

What does HTML stand for?

HyperText Markup Language! It's the skeleton of every webpage on the internet.

Why are some tags "self-closing"?

Tags like <img>, <br>, and <hr> don't wrap content — they just place something on the page, so they don't need a closing tag.
